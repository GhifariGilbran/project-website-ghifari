# Portfolio Website - Muhammad Ghifari Gilbran

Selamat datang di repositori kode sumber untuk website portfolio pribadi saya! Website ini dirancang untuk menampilkan profil, keterampilan, dan berbagai proyek yang pernah saya kerjakan.

## 🚀 Fitur Utama

- **Desain Modern (Glassmorphism):** Antarmuka pengguna (*User Interface*) dirancang dengan gaya modern menggunakan efek kaca tembus pandang (*blur*), gradasi warna, dan bayangan yang elegan.
- **Responsif (Mobile-Friendly):** Tampilan website beradaptasi dengan sempurna pada berbagai ukuran layar, mulai dari desktop hingga ponsel cerdas, lengkap dengan *hamburger menu* untuk navigasi di perangkat mobile.
- **Animasi Dinamis:** Dilengkapi dengan efek partikel interaktif pada latar belakang, efek *reveal on scroll* (elemen muncul saat di-scroll), dan *hover states* yang mulus.
- **Skill Switcher Otomatis:** Menampilkan keahlian utama (Web Development, Data Analyst, Pemrograman) secara bergantian dengan transisi yang halus.

## 🛠️ Teknologi yang Digunakan

Website ini dibangun menggunakan teknologi dasar web (Vanilla) untuk memastikan performa yang ringan dan cepat tanpa bergantung pada kerangka kerja (*framework*) yang berat:

- **HTML5:** Struktur semantik halaman web.
- **CSS3:** Penataan gaya, *layouting* (Flexbox/Grid), animasi, dan efek *glassmorphism*.
- **JavaScript (Vanilla):** Logika interaktif, animasi *canvas* (partikel), *skill switcher*, dan fungsionalitas menu navigasi.
- **FontAwesome:** Ikon vektor untuk bagian *skills*, *projects*, dan kontak sosial media.
- **Google Fonts:** Menggunakan *font* "Inter" untuk tipografi yang bersih dan modern.
